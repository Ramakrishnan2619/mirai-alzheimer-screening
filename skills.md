# MirAI Skills Demonstrated

## Backend Engineering
- **Flask** — REST API development with blueprints
- **SQLAlchemy** — ORM for SQLite database
- **Flask-JWT-Extended** — JWT token authentication
- **Flask-Bcrypt** — Secure password hashing
- **Flask-CORS** — Cross-origin resource sharing

## Machine Learning
- **XGBoost** — Tree-based gradient boosting classifier
- **scikit-learn** — KNNImputer for missing values, StandardScaler for normalization
- **Joblib** — Model serialization and loading
- **Pandas/NumPy** — Feature vector construction and data manipulation

## ML Pipeline Architecture
- **3-Stage Cascade Model** — Probability propagation between stages
- **Preprocessing Pipeline** — Imputer → Scaler → Model inference
- **Weighted Risk Fusion** — Multi-stage score aggregation

## Frontend Development
- **HTML5/CSS3** — Semantic markup and modern styling
- **JavaScript (ES6+)** — Async/await, fetch API, DOM manipulation
- **Bootstrap 5** — Responsive UI components
- **AOS.js** — Scroll animations

## Authentication & Security
- **JWT Sessions** — Stateless authentication tokens
- **bcrypt Hashing** — Secure password storage
- **Protected Routes** — Authorization middleware
- **Input Validation** — Server-side data validation

## Database Design
- **SQLite** — Lightweight relational database
- **User Model** — Authentication and profile data
- **Assessment Model** — Multi-stage screening data persistence
- **Relationships** — One-to-many user-assessment mapping

## DevOps & Deployment
- **Gunicorn** — Production WSGI server
- **Render.com** — Cloud deployment configuration
- **Environment Variables** — Configuration management
- **Git** — Version control

## Software Architecture
- **Modular Design** — Separation of concerns (routes, services, models)
- **Factory Pattern** — Flask application factory
- **Singleton Pattern** — Model loader caching
- **Blueprint Pattern** — Route organization
